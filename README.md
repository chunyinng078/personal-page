> A responsive personal portfolio website built to showcase my background and experiment with modern frontend UI libraries. 

It is live on: https://www.chunyinng078.site/

This project serves as a practical sandbox for me to explore and integrate various UI ecosystems within a **ReactJS** environment, focusing on clean architecture and rapid UI development.

### 🛠️ Tech Stack & Libraries
* **Core Framework:** ReactJS
* **Styling:** Tailwind CSS (for utility-first, responsive design)
* **UI Components:** daisyUI (for accelerating component building)
* **Icons:** Ant Design Icons (`antd icon`)
* **Deployment:** AWS Amplify (S3 + Route 53)
